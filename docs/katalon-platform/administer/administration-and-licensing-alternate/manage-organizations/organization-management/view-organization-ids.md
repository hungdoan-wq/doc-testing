# View Organization IDs
Learn how you can view an Organization's ID in Katalon TestOps.

:::tip requirements
You must be assigned the **Account Admin** role to perform this action.
:::

---

View an Organization's ID via the Organization Management tab in Organization.

## View an Organization's ID

1. Go to **Admin Settings > Organization**. You can find Admin Settings at the upper right corner of the page, indicated by a cog icon.

    The Organization Management tab appears by default.

2. Your Organization ID is displayed in the organization directory, within the name column. For example, in the image below, the Org ID is OG-4311XX for Account Admin.
