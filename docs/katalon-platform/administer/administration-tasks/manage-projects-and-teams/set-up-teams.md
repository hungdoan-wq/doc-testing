### <a id="task-3460" class="anchor_top_offset"/>Remove existing Users

<p xmlns="http://www.w3.org/1999/xhtml" className="shortdesc">Removing a user stops them from using their associated Katalon license(s) and allows you to optimize your resources for other users of your organization. </p> 
<section xmlns="http://www.w3.org/1999/xhtml" className="section context"><div className="note note note_note"><span className="note__title">Note:</span> <ul className="ul"><li className="li"><p className="p">You must be the Owner or Administrator of an Organization to do this. For more details, see: <a className="xref" href="/katalon-platform/administer/administration-tasks/manage-users#change-user-role">Change user role</a>.</p></li></ul></div><div className="note warning note_warning"><span className="note__title">Warning:</span> <ul className="ul"><li className="li">By clicking <span className="ph uicontrol">Remove</span>, you immediately terminate the current session that the user is working on with their online licenses in Katalon Studio. Perform this step with caution. </li><li className="li">Removing a Licensed User working with an offline license does not revoke the license.</li></ul></div><p className="p">To remove existing Users, do as follows:</p></section> 
<ol xmlns="http://www.w3.org/1999/xhtml" className="ol steps"><li className="li step stepexpand"><span className="ph cmd">In the <span className="ph uicontrol">Manage Users</span> page, click on the <em className="ph i">Trash bin</em> icon to remove a User.</span><div className="itemgroup info"><img className="image" width={850} src={useBaseUrl("/7fc3d070-4181-11ee-9d19-0242c7a41fd4/KT-aug2023-remove-existing-users.png")} alt="Remove existing users" /></div></li><li className="li step stepexpand"><span className="ph cmd">A <span className="ph uicontrol">Remove</span> dialog pops up. Confirm your action by clicking <span className="ph uicontrol">Remove</span>.</span></li></ol> 
<section xmlns="http://www.w3.org/1999/xhtml" className="section result">You have removed an existing user from your organization.</section>
