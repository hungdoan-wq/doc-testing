<p xmlns="http://www.w3.org/1999/xhtml" className="p">A user group is a  group of users in an Organization that shares the same purpose, goal, or characteristic. You can invite a user group to a Project at once, rather than inviting every single user's email.</p> 
<div xmlns="http://www.w3.org/1999/xhtml" className="note important note_important"><span className="note__title">Important:</span> <ul className="ul"><li className="li"><p className="p">You must be an Owner or Admins of your Organization.</p></li></ul></div>

## <a id="task-1414" class="anchor_top_offset"/>Create a user group

<section xmlns="http://www.w3.org/1999/xhtml" className="section context">To create a new user group, do as follows:</section> 
<ol xmlns="http://www.w3.org/1999/xhtml" className="ol steps"><li className="li step stepexpand"><span className="ph cmd">Sign in to <a className="xref j-external-link" href="https://my.katalon.com/" target="_blank">TestOps Admin</a>.</span></li><li className="li step stepexpand"><span className="ph cmd">From the Account admin page, select your Organization.</span></li><li className="li step stepexpand"><span className="ph cmd">Once you're in your Organization page, select <span className="ph uicontrol">User Groups</span> from the sidebar.</span><div className="itemgroup info"> The <span className="ph uicontrol">User Groups</span> page appears.<img className="image" width={800} src={useBaseUrl("/ef2e9530-db78-11ee-9719-0242c7a41fd4/KT-Choose_from_sidebar.png")} alt="User Groups page" /></div></li><li className="li step stepexpand"><span className="ph cmd">Click <span className="ph uicontrol">Create New Group</span> on the top-right corner of the <span className="ph uicontrol">User Groups</span> page.</span></li><li className="li step stepexpand"><span className="ph cmd">The <span className="ph uicontrol">Create New Group</span> dialog pops up as below.</span><div className="itemgroup info"><img className="image" width={500} src={useBaseUrl("/ef363650-db78-11ee-9719-0242c7a41fd4/KT-Create_new_group.png")} alt="Create New Group dialog" /></div><ol type="a" className="ol substeps"><li className="li substep substepexpand"><span className="ph cmd">Enter your desired group name.</span><div className="itemgroup info">The group name must be unique and should be easy-to-search. When you invite this group to a Project, you will need to enter this group name.</div></li><li className="li substep substepexpand"><span className="ph cmd">Select your users from this Organization and enter your group description (if any).</span></li></ol></li><li className="li step stepexpand"><span className="ph cmd">Click <span className="ph uicontrol">Create</span> once you're done.</span></li></ol> 
<section xmlns="http://www.w3.org/1999/xhtml" className="section result"><p className="p">You've created your user group successfully.</p> <p className="p">Now you can invite this user group to any Project. To learn how, see: <a className="xref" href="/katalon-platform/administer/administration-tasks/manage-projects-and-teams/set-up-your-projects#task-zd2n49av">Add users to a Project</a>.</p></section> 

## <a id="task-3650" class="anchor_top_offset"/>Update  a user group's information

<section xmlns="http://www.w3.org/1999/xhtml" className="section context">To view and update a user group's information, do as follows:</section> 
<ol xmlns="http://www.w3.org/1999/xhtml" className="ol steps"><li className="li step stepexpand"><span className="ph cmd">Go to the <span className="ph uicontrol">User Groups</span> page, then select your desired user group by clicking on the group name.</span><div className="itemgroup info"><img className="image" src={useBaseUrl("/ef0c1910-db78-11ee-9719-0242c7a41fd4/KT-User_group_list.png")} /></div></li><li className="li step stepexpand"><span className="ph cmd">The <span className="ph uicontrol">Group Detail</span> page displays your group's information as the following screenshot.</span><div className="itemgroup info"><img className="image" width={700} src={useBaseUrl("/ef68b800-db78-11ee-9719-0242c7a41fd4/KT-Group_details.png")} alt="Group details page" /></div></li><li className="li step stepexpand"><span className="ph cmd">You can edit your group name or group description, then click <span className="ph uicontrol">Update</span> to save changes.</span></li></ol>

## <a id="task-6477" class="anchor_top_offset"/>Remove existing users

<section xmlns="http://www.w3.org/1999/xhtml" className="section context">Follow these steps to remove the group's users:</section> 
<ol xmlns="http://www.w3.org/1999/xhtml" className="ol steps"><li className="li step stepexpand"><span className="ph cmd">Go to the <span className="ph uicontrol">User Groups</span> page and select your desired group. </span><div className="itemgroup stepresult">The <span className="ph uicontrol">Group Detail</span> page appears.</div></li><li className="li step stepexpand"><span className="ph cmd">Navigate to the <span className="ph uicontrol">Users</span> section, you can check the list of the group's users here.</span></li><li className="li step stepexpand"><span className="ph cmd">Click on the <em className="ph i">trash bin</em> icon next to the user that you want to remove, then click <span className="ph uicontrol">Remove</span> to confirm your action.</span><div className="itemgroup info"><img className="image" width={700} src={useBaseUrl("/ef823370-db78-11ee-9719-0242c7a41fd4/KT-_Groups_Users_list.png")} alt="Users section" /><div className="note note note_note"><span className="note__title">Note:</span> <ul className="ul"><li className="li">Removing users from this group will not affect their access to the relevant projects and features associated with this group.</li></ul></div></div></li></ol> 

## <a id="concept-1437" class="anchor_top_offset"/>Delete a user group

<div xmlns="http://www.w3.org/1999/xhtml" className="note note note_note"><span className="note__title">Note:</span> <ul className="ul"><li className="li">This action will not affect the membership status of users within your Organization or any Projects.</li></ul></div>
<p xmlns="http://www.w3.org/1999/xhtml" className="p">There are two (2) ways to delete a user group:</p> 
<div xmlns="http://www.w3.org/1999/xhtml" className="p"><ul className="ul"><li className="li"><p className="p">On the <span className="ph uicontrol">User Groups</span> page:</p><p className="p">You'll find the <em className="ph i">trash bin</em> icon next to each group.</p><img className="image" width={700} src={useBaseUrl("/ef3e9ac0-db78-11ee-9719-0242c7a41fd4/KT-_Delete_a_user_group.png")} alt="Delete group" /></li><li className="li"><p className="p">On the <span className="ph uicontrol">Group Detail</span> page:</p><p className="p">Scroll down til the bottom of the page, you'll find the <span className="ph uicontrol">Delete Group</span> section.</p><img className="image" width={800} src={useBaseUrl("/ef468a00-db78-11ee-9719-0242c7a41fd4/KT-Delete_group.png")} alt="Delete Group section" /></li></ul></div>
