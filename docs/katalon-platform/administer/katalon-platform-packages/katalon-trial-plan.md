# Katalon Trial Plan

<p xmlns="http://www.w3.org/1999/xhtml" className="p">This article provides an overview of the Katalon Trial Plan and what it encompasses, as well as how you can start using Katalon on a trial plan.</p> 

## Overview

<div xmlns="http://www.w3.org/1999/xhtml" className="p"><div className="note note note_note"><span className="note__title">Note:</span> <ul className="ul"><li className="li"><p className="p">Create a Katalon account <a className="xref j-external-link" href="https://login.katalon.com/realms/katalon/login-actions/registration?client_id=katalon-marketing&tab_id=zKOvze06dkQ" target="_blank">from our website</a>. </p></li></ul></div></div>

The Katalon Trial Plan is designed for new users. It offers a comprehensive introduction to Katalon's suite of tools through a 30-day trial period. This plan is automatically activated after you create and successfully verify your account and log in to Katalon Studio for the first time.

### Trial benefits

Upon your first login to Katalon Studio, new users will be granted the following for free:

**Katalon Trial Licenses**:
- One (1) Katalon Studio Enterprise (KSE) license - see: [Katalon Studio Enterprise per-user license](/katalon-studio/katalon-studio-enterprise-and-katalon-runtime-engine-license/katalon-studio-enterprise-per-user-license)
- One (1) Katalon Runtime Engine (KRE) license - see: [Katalon Runtime Engine floating license](/katalon-studio/katalon-studio-enterprise-and-katalon-runtime-engine-license/katalon-runtime-engine-floating-license)
- Three (3) [TestCloud sessions](/katalon-platform/administer/katalon-platform-packages/katalon-platform-quotas#testcloud-sessions)/parallel tests for Desktop & Mobile Browser Testing
- Unlimited [visual testing checkpoints](/katalon-platform/administer/katalon-platform-packages/katalon-platform-quotas#visual-testing-checkpoint-images)

**Katalon TestOps**:
For access to Katalon TestOps and its features, please contact our sales team for more information about available plans and pricing options.

For information about our subscription options and how to purchase them, see: [Katalon Pricing](https://katalon.com/pricing).

:::important
- Each trial license can be activated on only one machine at a time.
- Trial duration is 30 days.
- Each user can only use the trial once per account.
- Once your 30-day trial period expires, you have the option to continue using Katalon by subscribing to our paid plans or purchasing licenses/sessions.
:::

## Start your Katalon Studio Enterprise Trial

This tutorial focuses on how you can start your 30-day Katalon Studio Enterprise Trial.

### Prerequisites

- You have [created a Katalon account](https://login.katalon.com/realms/katalon/login-actions/registration?client_id=katalon-marketing&tab_id=zKOvze06dkQ).
- You have downloaded and installed Katalon Studio Enterprise. If you have not, refer to this guide: [Install Katalon Studio](/katalon-studio/get-started/install-katalon-studio)

## Steps to activate your trial plan

When you first log in to the Katalon Studio Enterprise application, the trial license associated with your account is automatically activated and valid for 30 days.

1. Open Katalon Studio Enterprise. The login dialog appears.

    <img width={400} src={useBaseUrl("/669e2490-e282-11ee-b3a4-0242c7a41fd4/ks-970-login-dialog.png")} alt="Studio login" />

    1. Click **Log in from Browser** to be redirected to the Katalon login page.

    2. Log in by using your preferred credentials.

4. Upon successful login, you are redirected to Katalon Studio Enterprise.

    The Studio Walkthrough dialog pops up displaying options to go on an onboarding tour for the type of test you wish to perform: Web UI, Mobile, API, Desktop, or Behavior-Driven Development (BDD). 

    <img width={800} src={useBaseUrl("/694fa640-d13a-11ee-9657-0242c7a41fd4/Studio_walkthrough.png")} alt="Studio walkthrough" />

    Alternatively, you can click **Close** and start to use KSE immediately.

You have begun your trial of Katalon Studio Enterprise. 

To check the expiration date, go to **Katalon Studio Enterprise > About Katalon Studio Enterprise**. The Expiration date indicates when you will have to upgrade to an Enterprise license. Visit our [pricing page](https://katalon.com/pricing) to learn more.

<img width={800} src={useBaseUrl("/2001f2ea-e133-4599-a35b-89eee0683f7d/KS_trial_Aug_6_2024.png")} alt="Katalon Studio homepage" />

Otherwise, you can continue with the free version of Katalon Studio. You will need to download and install Katalon Studio Free from our [website](https://katalon.com/).

View a comparison of Katalon Studio's free and enterprise features here: [Katalon Studio vs Katalon Studio Enterprise Features](/katalon-studio/katalon-studio-enterprise-and-katalon-runtime-engine-license/katalon-studio-vs-katalon-studio-enterprise-features).

## Start your Katalon Runtime Engine Trial

After the activation of your Katalon Runtime Engine (KRE) trial license, you will have full access to KRE for 30 days.

To begin, you need to authenticate your account with your Katalon API key. For more information, see: [API Keys](/katalon-platform/administer/profile/katalon-api-key-in-katalon-testops).

For detailed instructions on obtaining and using KRE, refer to this guide: [Get started with Katalon Runtime Engine](/katalon-studio/execute-tests/katalon-runtime-engine/get-started-with-katalon-runtime-engine).
