<p xmlns="http://www.w3.org/1999/xhtml" className="p">The Defects Report Slack notification integration automatically notifies users in a specific Slack channel whenever a project member performs a corresponding action using the Defects Report function from TestOps. This allows for better work visibility among project members and for quickly checking issues without having to navigate to them manually.</p> 

## <a id="task-5808" class="anchor_top_offset"/>Set up Slack notifications for defects

<div xmlns="http://www.w3.org/1999/xhtml" className="section prereq p"><ul className="ul"><li className="li"><p className="p">You must have integrated Slack with TestOps. To learn how, see: <a className="xref" href="/katalon-platform/integrations/slack-integration">Slack integration</a>.</p></li></ul></div>
<section xmlns="http://www.w3.org/1999/xhtml" className="section context">When a test result is linked to a defect either automatically through automation rules or manually by a project member, you can select options to be notified on Slack. To learn how to set it up, follow these steps: </section> 
<ol xmlns="http://www.w3.org/1999/xhtml" className="ol steps"><li className="li step stepexpand"><span className="ph cmd">Log in to TestOps and navigate to <span className="ph menucascade"><span className="ph uicontrol">Settings</span><abbr title="and then"> &gt; </abbr><span className="ph uicontrol">Integrations</span></span>.</span></li><li className="li step stepexpand"><span className="ph cmd">Select Jira/Xray either from the sidebar under ALM &amp; Test Management or search it in the search bar.</span></li><li className="li step stepexpand"><span className="ph cmd">Click on the <span className="ph uicontrol">Notifications</span> tab. </span><div className="itemgroup info"><img className="image" width={500} src={useBaseUrl("/be541990-97ed-11ee-b8c3-0242c7a41fd4/TO_Notifications_Tab_Jira_12_11_2023.png")} alt="The notifications tab in the jira xray integration section of Katalon TestOps." /></div></li></ol> 
<div xmlns="http://www.w3.org/1999/xhtml" className="li stepsection">Receive notifications for manually linked defects</div>
<ol xmlns="http://www.w3.org/1999/xhtml" className="ol steps" start={4}><li className="li step stepexpand"><span className="ph cmd">Click on the tick box next to <span className="ph uicontrol">Notify channel when a defect has been linked by project member</span>. </span><div className="itemgroup info"><img className="image" width={600} src={useBaseUrl("/be491d10-97ed-11ee-b8c3-0242c7a41fd4/TO_JiraXray_12_11_2023.png")} alt="The slack notification setting for manual linking." /></div></li><li className="li step stepexpand"><span className="ph cmd">Click <span className="ph uicontrol">Save</span>.</span></li></ol> 
<div xmlns="http://www.w3.org/1999/xhtml" className="li stepsection">Receive notifications for linked defects via automation rules</div>
<ol xmlns="http://www.w3.org/1999/xhtml" className="ol steps" start={6}><li className="li step stepexpand"><span className="ph cmd">Tick <span className="ph uicontrol">Notify channel when a defect has been linked by automation rule</span>. </span><div className="itemgroup info"><img className="image" width={600} src={useBaseUrl("/be3d8450-97ed-11ee-b8c3-0242c7a41fd4/TO_Notifications_automation_rule_12_11_2023.png")} alt="The slack notification setting for automatic linking." /></div></li><li className="li step stepexpand"><span className="ph cmd">Click Save.</span></li></ol> 
<section xmlns="http://www.w3.org/1999/xhtml" className="section result"> <p className="p">You have set up Slack notifications for your linked defects.</p><div className="p">It contains the following information:<ul className="ul"><li className="li"><span className="ph uicontrol">Test Result ID</span></li><li className="li"><span className="ph uicontrol">Test Result Name</span></li><li className="li"><span className="ph uicontrol">Execution Name</span></li><li className="li"><span className="ph uicontrol">Status</span></li><li className="li">General information about the linked defects.</li></ul></div><p className="p">Click on any of the links to view the full details on TestOps. <img className="image" width={500} src={useBaseUrl("/be1f26e0-97ed-11ee-b8c3-0242c7a41fd4/TO_Slack_Notification_12_11_2023.png")} alt="A sample of a notification from TestOps on Slack." /></p></section>
