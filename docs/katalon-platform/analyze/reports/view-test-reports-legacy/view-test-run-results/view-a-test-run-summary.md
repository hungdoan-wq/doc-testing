<section xmlns="http://www.w3.org/1999/xhtml" className="section context">The test run <span className="ph uicontrol">Summary</span> tab provides you with an overview of a test run result. You may view statistics of all test cases executed in a test run, its execution history, as well as provide you with a quick view of failed test cases associated with the selected test run.</section>

It provides an overview of all your failed test results and a short summary of them, all of which you can resolve easily by using the Defects Report function. </span><div className="itemgroup info"><img className="image" width={600} src={useBaseUrl("/571dff70-bf1b-11ee-ac6d-0242c7a41fd4/TO_Failed_Test_Results.png")} alt="The failed test results section." /></div><ol type="a" className="ol substeps"><li className="li substep"><span className="ph cmd">Click on the <span className="ph uicontrol">Need Review</span> button to filter out results  with the statuses Failed, Error, or Incomplete that have not been linked with any defects yet. Learn more about linking using defects report here: <a className="xref" href="/katalon-platform/integrations/jira-integration/link-test-runs-to-jira-defects-in-katalon-testops">Link test runs  to Jira defects in Katalon TestOps</a>. </span></li><li className="li substep"><span className="ph cmd">Hover over an error message summary, shown as red text, to view its corresponding error message entirely. </span></li><li className="li substep"><span className="ph cmd">Click on the <span className="ph uicontrol">bug icon</span> to go to Defects Report. The fluctuation trend in test execution time may indicate flakiness or lack of reliability in your test scripts. <img className="image" width={600} src={useBaseUrl("/c9cd7c70-901a-11ed-998d-0242cfbc79b5/TO-Reports-Previous_Test_Runs.png")} alt="Previous test runs" /></span></li><li className="li step stepexpand"><div className="note note note_note"><span className="note__title">Note:</span> <ul className="ul"><li className="li">           <p className="p">The Sessions section is only available if you have run a configuration with an agent.</p>         </li></ul></div><span className="ph cmd">Scroll down to the <span className="ph uicontrol">Sessions</span> section to view session logs.
