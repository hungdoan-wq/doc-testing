# Create a Visual Baseline Collection

This document shows you how to create a visual baseline collection.

---
A baseline image is a reference image initially configured to compare with other image output from visual testing.

## Create a Visual Baseline Collection

1. Go to **Assets** > **Visual Baseline Collections**. The page lists all your collections. 

2. Click the **Create** button at the upper right corner. The Create Baseline Collection window appears.

<img src="https://tw-cdn.katalon.com/katalon-testops/Assets/Create Baseline Connection.png" alt="Create Visual Baseline Collection" width="500"/>

3. Input a name for your collection.

4. Upload a Baseline Image. 

5. Click **Create**.

#### Result

A notification confirms that you have successfully created a visual baseline collection.

---

## Upload a Baseline Image

1. Go to **Assets** > **Visual Baseline Collections** and click on the collection you want to add to.

2. Click **Upload Image** at the upper right corner.

<img src="https://tw-cdn.katalon.com/katalon-testops/Assets/Create Baseline Collection Upload Image 2.png" alt="Upload Image" width="1080"/>

3. Select the image you want to upload. <br/> Uploading an image will increase one version of the collection.

4. Click **Upload**.

#### Result 

A notification confirms that you have successfully uploaded a baseline image.
