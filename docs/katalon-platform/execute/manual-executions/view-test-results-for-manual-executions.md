---
title: View execution details
---

This shows you how to view execution details for manual test runs.

## Prerequisites

- Make sure you have the Test Lead or Tester role. Refer to this document for more details: [Project level roles](/katalon-platform/administer/administration-and-licensing-alternate/about-administration#project-level-roles).

---

## View the Execution Overview
<!-- https://katalon.atlassian.net/browse/TO-2273-->
Here you can view overall test run information such as test results across configurations or test run metadata. 

1. Go to **Executions**. The Executions list appears by default.

2. Click within the row of your chosen manual execution.

### Results
This brings you to its Execution Overview page.

<br/>
<img src="https://tw-cdn.katalon.com/katalon-testops/Execution/Manual Executions/Executions Execution Overview.png" alt="TO3 Execution Overview" width="1080"/>
<br/>

From here, you can:
- Link an execution to an Iteration via the **Link Iteration** button
- Open Testpak to view an execution's full steps and details via the **Open TestPak** button

---

## View report for completed test runs

1. Go to **Executions**. The Executions list appears by default.

2. Within the row of your chosen manual execution, click on the **View Report** button found in the right-hand side of the screen.

<br/>
<img src="https://tw-cdn.katalon.com/katalon-testops/Execution/Manual Executions/Executions View Report Button.png" alt="TO3 View Report Button" width="1080"/>
<br/>

Alternatively, you can also click on the **View Report** button found in the **Execution Overview** page.

<br/>
<img src="https://tw-cdn.katalon.com/katalon-testops/Execution/Manual Executions/Executions Execution Overview View Report Button.png" alt="TO3 Execution Overview View Report Button" width="1080"/>
<br/>

### Results

This brings you to its Report page where all information about the test run is displayed.

<br/>
<img src="https://tw-cdn.katalon.com/katalon-testops/Execution/Manual Executions/Executions View Report Test Run Page.png" alt="TO3 View Report Test Run Page" width="1080"/>

## View test results during execution
<!--https://katalon.atlassian.net/browse/TO-4210-->

This allows you to quickly review your test run information so you can gauge the progress of your execution.

1. If you're not already in TestPak, go to **Executions**. The Executions list appears by default.

2. Click within the row of your chosen manual execution.

3. From the **Execution Overview** page, click on **Open TestPak**.

<br/>
<img src="https://tw-cdn.katalon.com/katalon-testops/Execution/Manual Executions/Executions Execution Overview Open TestPak Button.png" alt="TO3 Execution Overview Open TestPak Button" width="1080"/>
<br/>

4. From within TestPak, click on the **View Results** button in the upper right corner of the screen.

<br/>
<img src="https://tw-cdn.katalon.com/katalon-testops/Execution/Manual Executions/Executions TestPak View Results Button Censored.png" alt="TO3 TestPak View Results Button" width="1080"/>
<br/>

### Results

The **Test Case Results** dialog pops up, granting a complete overview.

<br/>
<img src="https://tw-cdn.katalon.com/katalon-testops/Execution/Manual Executions/Executions TestPak View Test Case Results.png" alt="TO3 TestPak View Test Case Results" width="1080"/>
