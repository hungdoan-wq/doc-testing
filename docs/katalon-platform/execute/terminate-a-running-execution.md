<p xmlns="http://www.w3.org/1999/xhtml" className="shortdesc"> </p> 
<section xmlns="http://www.w3.org/1999/xhtml" className="section context">Terminate a running execution from its overview drawer or from the executions list. </section> 
<ol xmlns="http://www.w3.org/1999/xhtml" className="ol steps"><li className="li step"><span className="ph cmd">Go to Executions to <a className="xref" href="/katalon-testops/manage-executions/view-executions-list">view the executions list</a>. </span></li></ol> 
<div xmlns="http://www.w3.org/1999/xhtml" className="li stepsection">Terminate from Executions: </div>
<ol xmlns="http://www.w3.org/1999/xhtml" className="ol steps" start={2}><li className="li step"><span className="ph cmd">Within the row of the test run you'd like to terminate, click on the <span className="ph uicontrol">Stop</span> icon at the rightmost column.</span></li></ol> 
<div xmlns="http://www.w3.org/1999/xhtml" className="li stepsection">Terminate from the overview drawer:</div>
<ol xmlns="http://www.w3.org/1999/xhtml" className="ol steps" start={3}><li className="li step"><span className="ph cmd">Click on the row of a test run to open a drawer containing an overview of its information. </span></li><li className="li step"><span className="ph cmd">Click on the <span className="ph uicontrol">Terminate</span> button at the upper right corner. </span></li><li className="li step"><span className="ph cmd">A dialog box asks you to confirm your decision. Click <span className="ph uicontrol">Terminate</span>. </span></li></ol> 
<section xmlns="http://www.w3.org/1999/xhtml" className="section result">A toast notification confirms that you have terminated an execution. Its status also changes to Terminate. </section>
