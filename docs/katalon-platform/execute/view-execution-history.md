<p xmlns="http://www.w3.org/1999/xhtml" className="p">The <span className="ph uicontrol">History</span> page provides you with the list of all test runs triggered from <span className="ph uicontrol">TestOps</span>. This interface helps you track the running sessions and quickly search your test runs by their properties. </p> 
<p xmlns="http://www.w3.org/1999/xhtml" className="p">To view execution history, open your project in <span className="ph">TestOps</span> and go to <span className="ph uicontrol">Execution</span> &gt; <span className="ph uicontrol">History</span>.</p> 
<img xmlns="http://www.w3.org/1999/xhtml" className="image" width={700} src={useBaseUrl("/dd311b70-b9bb-11ee-ac6d-0242c7a41fd4/testops_execution_history.png")} alt="TestOps execution history page" /> 
<div xmlns="http://www.w3.org/1999/xhtml" className="p">The search and filter utility allows you to search for the test runs by their names, and filter by the following properties:<ul className="ul"><li className="li">Time frame</li><li className="li">Status</li><li className="li">Executor (the team member that triggered the execution)</li></ul></div>
<img xmlns="http://www.w3.org/1999/xhtml" className="image" width={700} src={useBaseUrl("/99ad8650-bfeb-11ee-ac6d-0242c7a41fd4/testops_execution_history_search_filter_utility.png")} /> 

## View a test run summary

<p xmlns="http://www.w3.org/1999/xhtml" className="p">Select a test run to open the summary panel with further information about failed test results by each test case and the running sessions categorized by status. </p> 
<p xmlns="http://www.w3.org/1999/xhtml" className="p"><img className="image" width={700} src={useBaseUrl("/940accb0-b9bc-11ee-ac6d-0242c7a41fd4/testops_execution_history_test_run_summary.png")} alt="Execution history test run summary" /></p> 

## Re-run, terminate, or delete test runs

<p xmlns="http://www.w3.org/1999/xhtml" className="p">Once a test run is completed, you can click the <em className="ph i">three-dot</em> icon and choose to re-run or delete that test run.<img className="image" width={700} src={useBaseUrl("/999e9230-bfeb-11ee-ac6d-0242c7a41fd4/TestOps_execution_history_threedot.png")} /></p> 
<p xmlns="http://www.w3.org/1999/xhtml" className="p">To terminate specific test runs, you can select individual test runs or select all, then click <span className="ph uicontrol">Terminate</span>. You can also terminate a test run by clicking on the stop button.<img className="image" width={700} src={useBaseUrl("/99b12fd0-bfeb-11ee-ac6d-0242c7a41fd4/TestOps_execution_history_select_execution.png")} /></p>
