---
title: "Katalon Studio Integration"
---

This document shows you how to upload test reports from Katalon Studio to Katalon TestOps.

## Upload test report automatically
To upload Katalon Studio's test reports automatically after executing a Test Suite or Test Suite Collection:

1. In Katalon Studio, go to **Project > Settings > Integration > Katalon TestOps**.
2. Enable the **Automatically submit test run result** option.

<img src="https://tw-cdn.katalon.com/katalon-platform/integration/ks-auto.png" alt="Upload test reports automatically in KS" width="700"/>


## Upload test reports manually
To upload Katalon Studio's test reports manually:

1. In Test Report detail views, click **Katalon TestOps > Upload**.

<img src="https://tw-cdn.katalon.com/katalon-platform/integration/ks-manual.png" alt="Upload test reports manually in KS" width="700"/>

2. Once upload successfully, go to **Katalon TestOps > Reports** to view a detailed analysis of test execution reports.
