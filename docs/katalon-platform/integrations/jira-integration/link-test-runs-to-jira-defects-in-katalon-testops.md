<p xmlns="http://www.w3.org/1999/xhtml" className="p">After executing a test run  in <span className="ph">Katalon TestOps</span>, you can link test results to an issue in Jira  directly by using <span className="ph uicontrol">Report Defect</span>.</p> 

## Requirements

<ul xmlns="http://www.w3.org/1999/xhtml" className="ul"><li className="li"><p className="p">You have configured your Jira integration. See: <a className="xref" href="/katalon-platform/integrations/jira-integration/enable-testops---jira-integration-for-test-management">Enable TestOps - Jira integration for test management</a>.</p></li></ul> 

## <a id="task-6716" class="anchor_top_offset"/>Link test results to Jira

<section xmlns="http://www.w3.org/1999/xhtml" className="section context">To link a failed test result to a Jira issue, follow these steps:</section> 
<ol xmlns="http://www.w3.org/1999/xhtml" className="ol steps"><li className="li step stepexpand"><span className="ph cmd">From your project, go to <span className="ph uicontrol">Reports</span> &gt; <span className="ph uicontrol">Test Runs</span>.</span></li><li className="li step stepexpand"><span className="ph cmd">Select a failed test run by clicking on its ID, then click on the <span className="ph uicontrol">Test Results</span> tab.<img className="image" width={400} src={useBaseUrl("/d7244060-776a-11ee-8403-0242c7a41fd4/TO_Defect_Test_Run_ID_October_2023_2.png")} alt="Test run ID in Katalon TestOps' reports page." /></span><div className="itemgroup stepresult">       <p className="p">The <span className="ph uicontrol">Test Results</span> page appears as below.</p>       <p className="p"><img className="image" width={700} src={useBaseUrl("/d76037f0-776a-11ee-8403-0242c7a41fd4/TO_Test_Results_Page_ID44.png")} alt="The Test Results page in Katalon TestOps." /></p>     </div></li><li className="li step stepexpand"><span className="ph cmd">Navigate to and click on the <span className="ph uicontrol">Report Defect</span> button at the rightmost column. It has the bug icon.</span><ol type="a" className="ol substeps"><li className="li substep substepexpand"><span className="ph cmd">Alternatively, you can report a defect by going to a specific test result's detail page.To do so, From the <span className="ph uicontrol">Test Results</span> page, navigate to the <span className="ph uicontrol">Failed Test Results</span> section.</span><div className="itemgroup info"><img className="image" width={700} src={useBaseUrl("/d7575e50-776a-11ee-8403-0242c7a41fd4/TO_Failed_Test_Result_Section_October_2023.png")} /></div></li><li className="li substep substepexpand"><span className="ph cmd">Click on the ID of the test result you want to connect to a Jira issue. This brings you to its detail page.</span></li><li className="li substep substepexpand"><span className="ph cmd">Click on <span className="ph uicontrol">Report Defect</span> at the upper right corner. </span><div className="itemgroup info"><img className="image" src={useBaseUrl("/d74f4800-776a-11ee-8403-0242c7a41fd4/TO_Report_Defect_Button_Specific_Test_Result_October_2023.png")} alt="Click on the Report Defect Button at the test result's page." /></div></li></ol><div className="itemgroup stepresult">A pop-up message box appears. It has your current Jira project selected and is set to <span className="ph uicontrol">Link existing issue</span> by default.<p className="p"><img className="image" width={400} src={useBaseUrl("/d72c56b0-776a-11ee-8403-0242c7a41fd4/TO_Report_Defect_Pop_Up_October_2023.png")} alt="The Report Defect pop up in Katalon TestOps." /></p>     </div></li><li className="li step stepexpand"><span className="ph cmd">Choose from the suggested defects list, which presents three options of recently linked defects with similar failures as the current test result.</span><div className="itemgroup info"> <img className="image" width={500} src={useBaseUrl("/180e73b0-ae49-11ee-b8c3-0242c7a41fd4/TO_January_10_2024_Suggested_Defects_2.png")} alt="Suggested defects within the Report Defect dialog box." /></div><div className="itemgroup info">Alternatively, you could also search for the Jira issue you want to connect the report to from the dropdown menu. Then click <span className="ph uicontrol">Link Defect</span>.</div><div className="itemgroup info"><img className="image" width={500} src={useBaseUrl("/d74758c0-776a-11ee-8403-0242c7a41fd4/TO_Report_Defect_Link_Existing_Issue_October_2023_2.png")} alt="Other linked defects shown in the dropdown within the Report Defect dialog box." /></div></li></ol> 
<section xmlns="http://www.w3.org/1999/xhtml" className="section result">You have successfully linked a defect test result to a Jira issue. A notification will pop up at the upper right corner confirming this. <p className="p"><img className="image" width={400} src={useBaseUrl("/d7350940-776a-11ee-8403-0242c7a41fd4/TO_Defect_Report_Done.png")} alt="You have connected a defect report to Jira." /></p><p className="p">Your issue also shows under the Defects column when viewing the test run's summary. </p><p className="p"><img className="image" width={500} src={useBaseUrl("/d71af190-776a-11ee-8403-0242c7a41fd4/TO_Defects_Column__Jira_Issues_Show_Ocotber_2023_2.png")} alt="The Defects column shows your linked Jira issues." /></p><p className="p">You can automatically be notified in Slack when this action is performed by setting up an integration. See: <a className="xref" href="/katalon-platform/analyze/analytics/defects-report/defects-report-slack-notifications">Defects Report Slack notifications</a>.</p></section> 

### <a id="task-262" class="anchor_top_offset"/>Quick tips

<section xmlns="http://www.w3.org/1999/xhtml" className="section context">   <p className="p">We recommend you install the Katalon Integration Plugin in your Chrome browser. This plugin allows you to navigate directly to Jira to create a new Jira issue. It then automatically links the same Jira issue back to the Test Result you have selected. To install it, follow these steps:</p> </section> 
<ol xmlns="http://www.w3.org/1999/xhtml" className="ol steps"><li className="li step stepexpand"><span className="ph cmd">Install this extension from the Chrome web store (<a className="xref j-external-link" href="https://chrome.google.com/webstore/detail/katalon-integration/cechonbcopffiimhnkgghckbgipciedg" target="_blank">Katalon Integration</a>) by clicking on <span className="ph uicontrol">Add to Chrome</span>.</span><div className="itemgroup info"><img className="image" width={700} src={useBaseUrl("/ffa20e40-9845-11ee-b8c3-0242c7a41fd4/TO_quick_tips_1.png")} alt="The Katalon integration in the chrome store" /></div></li><li className="li step stepexpand"><span className="ph cmd">A dialog box asks you to confirm the installation. Click on <span className="ph uicontrol">Add extension</span>.</span><div className="itemgroup info"><img className="image" width={500} src={useBaseUrl("/ff987150-9845-11ee-b8c3-0242c7a41fd4/TO_quick_tips_add_extension.png")} alt="Dialog box asks you to confirm the addition of the extension." /></div></li><li className="li step stepexpand"><span className="ph cmd">A pop up notification tells you that the Katalon Integration has been added to Chrome. </span><div className="itemgroup info"><img className="image" width={500} src={useBaseUrl("/ff630970-9845-11ee-b8c3-0242c7a41fd4/TO_quick_tips_has_been_added_to_chrome.png")} alt="Katalon integration has been added to chrome." /></div></li><li className="li step stepexpand"><span className="ph cmd">You can manage your extension by clicking on the Extensions icon beside the search bar or by accessing it via Chrome's settings menu.<img className="image" width={500} src={useBaseUrl("/ff8dea00-9845-11ee-b8c3-0242c7a41fd4/Chrome_manage_extensions.png")} alt="Manage your extensions in Google Chrome." /></span></li><li className="li step stepexpand"><span className="ph cmd">After installation, go back and login to your TestOps account. Then, click on <span className="ph menucascade"><span className="ph uicontrol">Reports</span><abbr title="and then"> &gt; </abbr><span className="ph uicontrol">Test Runs</span><abbr title="and then"> &gt; </abbr><span className="ph uicontrol">Test Runs</span></span>.</span></li><li className="li step stepexpand"><span className="ph cmd">In the <span className="ph uicontrol">All Test Runs</span> section, click on the ID of one of your test runs. </span><ol type="a" className="ol substeps"><li className="li substep"><span className="ph cmd">This opens the Summary page. Scroll down to the <span className="ph uicontrol">Failed Test Results</span> section and click on a test result ID. <img className="image" width={500} src={useBaseUrl("/ff6f8c90-9845-11ee-b8c3-0242c7a41fd4/TO_Click_on_test_run_ID.png")} alt="The test run ID in the All Test Runs section." /></span></li><li className="li substep"><span className="ph cmd">This brings you to the <span className="ph uicontrol">Result</span> page. </span></li></ol></li><li className="li step stepexpand"><span className="ph cmd">Click on the Katalon Plugin button.</span><div className="itemgroup info"><img className="image" width={500} src={useBaseUrl("/ff7a13e0-9845-11ee-b8c3-0242c7a41fd4/TO_Katalon_plugin_button.png")} alt="The Katalon Plugin button in Katalon TestOps." /></div><ol type="a" className="ol substeps"><li className="li substep"><span className="ph cmd">This opens your connected Jira board. You can now work quickly between Jira and TestOps.</span></li></ol></li><li className="li step stepexpand"><span className="ph cmd">Use the plugin icon to switch back to TestOps when needed.</span><div className="itemgroup info"><img className="image" width={500} src={useBaseUrl("/ff83fef0-9845-11ee-b8c3-0242c7a41fd4/Jira_Katalon_plugin.png")} alt="Katalon plugin button in Jira" /></div></li></ol> 
<section xmlns="http://www.w3.org/1999/xhtml" className="section result">The new Jira issue is automatically linked to the Test Result you have selected.</section> 

## <a id="task-2643" class="anchor_top_offset"/>View linked test runs in Jira

<section xmlns="http://www.w3.org/1999/xhtml" className="section context">To view linked test runs in Jira, do as follows:</section> 
<ol xmlns="http://www.w3.org/1999/xhtml" className="ol steps"><li className="li step stepexpand"><span className="ph cmd">Go to the Jira issue you have linked the Test Results to.</span></li><li className="li step stepexpand"><span className="ph cmd">Click on the <span className="ph uicontrol">Linked Test Runs</span> section to see all test results of the linked test runs.</span><div className="itemgroup info"><img className="image" src={useBaseUrl("https://github.com/katalon-studio/docs-images/raw/master/katalon-analytics/docs/testops-revamp-july-ka-defects/jira-link-test-run-section-plugin.png")} alt="jira linked test runs section" /><br /><br /></div></li></ol>
