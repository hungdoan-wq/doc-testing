<div xmlns="http://www.w3.org/1999/xhtml" className="section prereq p"><ul className="ul"><li className="li">You must be the Organization Owner or Admin.</li><li className="li">You have linked your Jira test cases to the Jira test plan.</li><li className="li">You have linked a Katalon test run to an Xray test plan.</li></ul></div>
<section xmlns="http://www.w3.org/1999/xhtml" className="section context"><p className="p">If you have chosen the <span className="ph uicontrol">Manual push only</span> mechanism to upload Katalon test result to Jira/Xray, Katalon will not automatically upload all test results to Jira/Xray. Instead, you can decide which test run results to upload to Jira/Xray.</p><p className="p">For a demonstration, refer to this video: <a className="xref j-external-link" href="https://screencast-o-matic.com/watch/c0VtfQVxX6q" target="_blank">Pushing results to Xray</a>.</p><p className="p">Additionally, you can use the Report Uploader with the parameter <code className="ph codeph">--push-to-xray=true</code>: <a className="xref" href="/katalon-platform/analyze/reports/upload-test-reports/use-katalon-report-uploader">Use Katalon Report Uploader</a>.</p><p className="p">Here's a step-by-step on how to push test results to Xray in TestOps:</p></section>

className="itemgroup stepresult">The TestOps Test Runs page appears as below. <img className="image" src={useBaseUrl("/d0ff1c10-75df-11ed-a602-0242cfbc79b5/TO-Reports-Test_Runs-Test_Runs.png")} stepexpand"><span className="ph cmd">From the test run detail page, select <span className="ph uicontrol">Push to Xray</span>.</span><div className="itemgroup info"><img className="image"
