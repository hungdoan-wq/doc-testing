<p xmlns="http://www.w3.org/1999/xhtml" className="shortdesc">You can upload tests from <span className="ph">Katalon Studio</span> directly to <span className="ph">Katalon TestOps</span> via a ZIP file upload. This makes it easy to leverage features like scheduling or executing test runs and viewing reports without having to configure a git repository. </p> 
<section xmlns="http://www.w3.org/1999/xhtml" className="section context">You can upload a ZIP file of your test directly to <span className="ph">TestOps</span> by accessing <span className="ph uicontrol">Script Repositories</span> within <span className="ph uicontrol">Settings</span>. To learn how to do so step by step, follow these instructions:</section> 
<ol xmlns="http://www.w3.org/1999/xhtml" className="ol steps"><li className="li step stepexpand"><span className="ph cmd"><a className="xref j-external-link" href="https://my.katalon.com/" target="_blank">Sign in to Katalon TestOps</a> and go to your project. </span></li><li className="li step stepexpand"><span className="ph cmd">Click on <span className="ph menucascade"><span className="ph uicontrol">Settings</span><abbr title="and then"> &gt; </abbr><span className="ph uicontrol">Script Repositories</span></span>. Select the ZIP you would like to upload. </span><div className="itemgroup info"><img className="image" width={700} src={useBaseUrl("/3d1ab490-b414-11ee-ac6d-0242c7a41fd4/TO_Upload_Script_Repositories_Window_File_January_16_2024_2.png")} />       <div className="note note note_note"><span className="note__title">Note:</span>          <ul className="ul"><li className="li">             <p className="p">The maximum file size limit is 300MB.</p>           </li></ul>       </div>     </div></li><li className="li step stepexpand"><span className="ph cmd">The file you selected shows next to the <span className="ph uicontrol">Choose File</span> button. ## <a id="task-3794" class="anchor_top_offset"/>Manage your Script Repository

<section xmlns="http://www.w3.org/1999/xhtml" className="section context">You can <span className="ph uicontrol">Edit</span> the details of your repository or <span className="ph uicontrol">Delete</span> it from your project by clicking on the corresponding buttons at the upper right corner. </section> 
<ol xmlns="http://www.w3.org/1999/xhtml" className="ol steps"><li className="li step stepexpand"><span className="ph cmd">Editing your repository allows you to change aspects of it like its name, description, setting as the default, or the ZIP file itself. </span></li><li className="li step stepexpand"><span className="ph cmd">Click <span className="ph uicontrol">Edit</span>. This brings you back to the upload page where you can make changes as needed. </span><div className="itemgroup info"><img className="image" width={850} src={useBaseUrl("/3d35b6a0-b414-11ee-ac6d-0242c7a41fd4/TO_Script_Repositories_ZIP_Uploaded_Details_Edit_Annotated_January_16_2024.png")} alt="The Edit button in the Script Repository page." /></div></li><li className="li step stepexpand"><span className="ph cmd">Once done, click <span className="ph uicontrol">Save</span>. </span></li><li className="li step stepexpand"><span className="ph cmd">To delete your repository, click on <span className="ph uicontrol">Delete</span>. A dialog box pops up asking you to confirm your decision. Click <span className="ph uicontrol">Delete</span> again. </span><div className="itemgroup info"><div className="note warning note_warning"><span className="note__title">Warning:</span> <ul className="ul"><li className="li"><p className="p">You cannot delete a Script Repository that is currently in use. </p></li></ul></div></div><div className="itemgroup info"><img className="image" width={850} src={useBaseUrl("/3d4a7720-b414-11ee-ac6d-0242c7a41fd4/TO_Script_Repositories_ZIP_Uploaded_Details_Delete_Annotated_January_16_2024.png")} alt="The Delete button in the Script Repository page." /></div></li></ol> 
<section xmlns="http://www.w3.org/1999/xhtml" className="section result">You have successfully edited or deleted your Script Repository.</section>
