title: Install plugins online from Katalon Store
---
import useBaseUrl from '@docusaurus/useBaseUrl';



<p xmlns="http://www.w3.org/1999/xhtml" className="shortdesc">This article shows you how to install plugins online from the Katalon Store.</p> 
<div xmlns="http://www.w3.org/1999/xhtml" className="section prereq p"><ul className="ul"><li className="li">Katalon Studio version 7.0.0 onwards.</li><li className="li">An active  Katalon Studio Enterprise license.</li></ul></div>
<ol xmlns="http://www.w3.org/1999/xhtml" className="ol steps"><li className="li step stepexpand"><span className="ph cmd">To access the <span className="ph">Katalon Store</span> via <span className="ph uicontrol">Katalon         Studio</span>, click on the dropdown       next to the <em className="ph i">Profile</em> icon and select <span className="ph uicontrol">Visit Plugin         Store</span>.</span><div className="itemgroup info"><img className="image" width={300} src={useBaseUrl("/df2863a0-6c6b-11ed-a602-0242cfbc79b5/ks-visit-plugin-store.png")} alt="Visit Plugin Store" /></div><div className="itemgroup stepresult">You are directed to the Katalon Store website.</div></li><li className="li step stepexpand"><span className="ph cmd">In the Katalon Store homepage, browser for the plugin you want to install,  or type the plugin name in the search bar.</span><div className="itemgroup info"><img className="image" width={700} src={useBaseUrl("/dc6c46e0-1100-4f83-9815-f6a09edf0768/katalon-store-search-plugin.png")} alt="Katalon Store homepage" /></div></li><li className="li step stepexpand"><span className="ph cmd">Select the thumbnail image of the plugin, and in the product details page click <span className="ph uicontrol">Install</span> then click <span className="ph uicontrol">I Agree</span> in the pop-up dialog.</span><div className="itemgroup info"><img className="image" width={300} src={useBaseUrl("/63d71ab5-49e0-44e7-a9ce-ae84b8843687/katalon-store-agree-plugin.png")} /></div><div className="itemgroup stepresult">Once the plugin is installed, a <em className="ph i">Plugin Successfully Installed!</em> message displays accordingly.<p className="p"><img className="image" width={400} src={useBaseUrl("/839b07d0-644f-4fec-bd27-54cfbb452467/plugin_installed_message.png")} /></p></div></li><li className="li step stepexpand"><span className="ph cmd">In Katalon Studio, open the <span className="ph uicontrol">Profile</span> drop-down menu and select <span className="ph uicontrol">Reload Plugins</span>.</span><div className="itemgroup info"><img className="image" width={300} src={useBaseUrl("/f1288530-6be9-11ed-a602-0242cfbc79b5/ks-reload-plugins.png")} /></div><div className="itemgroup stepresult">The <span className="ph uicontrol">Plugins</span> dialog appears with a list of installed plugins, the plugin version, the link to review that plugin in <span className="ph">Katalon Store</span>, and the reload plugin status to let you know if the installation has succeeded or failed.<p className="p"><img className="image" width={600} src={useBaseUrl("/52479b94-9251-4919-8e29-2f73f1c17df7/reload-plugin-result.png")} /></p></div></li></ol> 
<section xmlns="http://www.w3.org/1999/xhtml" className="section result"><p className="p">After you reload plugins, newly installed plugins are automatically activated. If you install a Katalon Studio plugin, you can see the logo of the plugin you have installed in the main toolbar. Custom Keywords plugins do not have logos. For Custom Keywords plugins​, you can find a list of custom keywords ready for use added to the <span className="ph uicontrol">Keyword Browsers</span>​.</p></section>
