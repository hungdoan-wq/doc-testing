When we manually open a single chrome tab, we can see many Chrome processes in Task Manager.   <img className="image" src={useBaseUrl("/b0947390-7d62-11ee-8403-0242c7a41fd4/Windows_-_multiple_Chrome_browser_processes.png")} alt="multiple Chrome processes" /></p></section> 
<div xmlns="http://www.w3.org/1999/xhtml" className="bodydiv troubleSolution"><section className="section cause"><div className="p">This is  behavior is expected and explained by Google and  experts:<ul className="ul"><li className="li"><p className="p"><a className="xref j-external-link" href="https://www.urtech.ca/2018/07/solved-why-google-chrome-has-so-many-processes/" target="_blank">SOLVED: Why Google Chrome Has So Many Processes</a></p></li><li className="li"><p className="p"><a className="xref j-external-link" href="https://blog.chromium.org/2008/09/multi-process-architecture.html" target="_blank">Multi-process Architecture</a></p></li><li className="li"><p className="p"><a className="xref j-external-link" href="https://www.technipages.com/why-does-chrome-open-so-many-processes" target="_blank">Why Does Chrome Open So Many Processes? - Technipages</a></p></li><li className="li"><p className="p"><a className="xref j-external-link" href="https://www.youtube.com/watch?v=YZ6xXhs0EMk" target="_blank">Google Chrome using multiple processes is this normal and why does it do that</a></p></li></ul></div></section><section className="section remedy"><ol className="ol steps"><li className="li step stepexpand"><span className="ph cmd">It is possible to set up Google Chrome browser on  Windows 10 to create only a single process, even if you open multiple tabs on your computer. Right-click on the Google Chrome icon in your <span className="ph uicontrol">Start Menu</span> and then click on <span className="ph uicontrol">Properties</span> option. In case the Chrome icon is on the Taskbar of your computer, right-click on Chrome icon, right-click on Google Chrome and then click on <span className="ph uicontrol">Properties</span>.</span><div className="itemgroup info"><img className="image" src={useBaseUrl("/64882670-7d64-11ee-8403-0242c7a41fd4/Google_Chrome_-_Properties.png")} alt="google Chrome - Properties" /></div></li><li className="li step stepexpand"><span className="ph cmd">On the Chrome Properties screen, click on the <span className="ph uicontrol">Target</span> text box and scroll to the end of the line.
